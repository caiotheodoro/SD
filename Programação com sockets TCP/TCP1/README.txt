Como Compilar/Executar
# python3 ./server.py
# python3 ./client.py

Bibliotecas usadas
-socket     Esta biblioteca permite o uso de uma interface do socket BSD.
-threading  Esta biblioteca constrói interfaces do módulo _thread, de nível inferior.
-os         Esta biblioteca permite usar a funcionalidade dependente do sistema operacional.
-sys        Esta biblioteca fornece acesso a algumas variáveis utilizadas e a funções que interagem fortemente com o intérprete.
-hashlib    Esta biblioteca implementa uma interface de algoritmos de hash e digestão de mensagens seguros.

Exemplo de uso
-Para logar com um usuário valido.
# campiolo 
# so

-Para mostrar o local onde esta o terminal.
# PWD

-Para acessar um outro arquivo.
# CHDIR a

-Mostrar todos os arquivos nesse diretório e mostrar a quatidade.
# GETFILES

-Mostrar todos os diretórios nesse diretório e mostrar a quatidade.
# GETDIRS

-Para finalizar a conexão
# EXIT
